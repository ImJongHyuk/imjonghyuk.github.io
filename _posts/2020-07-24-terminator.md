# (개발 환경) 더 강력한 터미널 터미네이터 (terminator) 

## 설치 & 설정 방법
### ubuntu apt (advanced packaging tool) 이용한 간단한 설치.
```bash
!/bin/bash
sudo apt-get update;
sudo apt-get install terminator;
```

### 설정 변경 파일 경로
```bash
~/.config/terminator/config
```
