## 하위 스크립트의 환경변수 설정 값을 실행한 쉘에서 유지하는 방법.
```bash
bash env_set.sh   # 일반적인 방법. (스크립트가 끝나면 더 이상 전역 환경 변수를 유지하지 않는다.)
. env_set.sh      # 현재 쉘에서 스크립트를 실행하는 방법. (스크립트가 현재 쉘에서 실행되므로 스크립트가 끝나도 전역 환경 변수를 유지한다.)
```
